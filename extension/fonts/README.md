# OpenDyslexic Font Setup

This directory should contain the OpenDyslexic font file.

## Download Instructions

1. Visit: https://opendyslexic.org/
2. Download the font package
3. Extract and find `OpenDyslexic-Regular.otf`
4. Place it in this directory

## File Structure

```
extension/fonts/
├── OpenDyslexic-Regular.otf   # Download this file
├── OpenDyslexic.b64            # Generated by encode-font.js
└── README.md                   # This file
```

## Encoding the Font

After downloading the font:

```bash
# Navigate to extension directory
cd extension

# Run encoding script
node scripts/encode-font.js

# Inject into content script
node scripts/inject-font.js
```

## Alternative: Use CDN (Not Recommended)

While you could load the font from a CDN, embedding it as base64 ensures:
- No external network requests
- No CORS issues
- Works offline
- Faster load times
- No dependency on external services

## Troubleshooting

### Font file not found
- Ensure the file is named exactly: `OpenDyslexic-Regular.otf`
- Check it's in the `extension/fonts/` directory
- Verify file permissions

### Encoding fails
- Ensure Node.js is installed
- Check file is not corrupted (try re-downloading)
- Verify you have read permissions

### Font not applying in browser
- Check content_script.js has the base64 data (not PLACEHOLDER)
- Reload the extension in chrome://extensions/
- Check browser console for errors
- Verify the page has been reloaded after domain registration

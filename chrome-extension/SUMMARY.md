## 🎨 How It Works

### User Flow 1: Manual Toggle
```
User visits website
    ↓
Blue button appears (top-right)
    ↓
User clicks button
    ↓
Font changes to OpenDyslexic
Word beginnings become bold
Spacing increases
    ↓
User clicks again
    ↓
Text restores to original
```

### User Flow 2: Global Mode
```
User clicks extension icon
    ↓
Popup opens
    ↓
User toggles "Enable on All Sites"
    ↓
Visits any website
    ↓
Dyslexia mode active automatically
    ↓
Can still toggle off per-page
```

### User Flow 3: AdaptEd Integration (Optional)
```
User in AdaptEd webapp
    ↓
Clicks citation link
    ↓
bridge.js enables global mode
    ↓
External site opens
    ↓
Dyslexia mode active automatically
```

## 💡 Key Decisions Made

### 1. Keep Toggle Button
**Decision:** Keep your toggle button approach  
**Reason:** Better UX, more control, works everywhere  
**Trade-off:** Doesn't auto-activate from AdaptEd (but bridge.js provides this)

### 2. Add Global Mode
**Decision:** Add "Enable on All Sites" toggle in popup  
**Reason:** Best of both worlds - manual control + auto-activation  
**Implementation:** Popup UI + message passing

### 3. Embed Font
**Decision:** Embed OpenDyslexic as base64  
**Reason:** PRD requirement, works without system font  
**Implementation:** embed-font.js script

### 4. Add Popup
**Decision:** Create popup UI  
**Reason:** PRD requirement, provides settings interface  
**Implementation:** popup.html/css/js

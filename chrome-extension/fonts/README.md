# OpenDyslexic Font Setup

## Step 1: Download the Font

1. Visit: https://opendyslexic.org/
2. Click the "Download" button
3. Extract the ZIP file
4. Find `OpenDyslexic-Regular.otf`
5. Place it in this folder: `chrome-extension/fonts/OpenDyslexic-Regular.otf`

## Step 2: Encode the Font

```bash
cd chrome-extension
node embed-font.js
```

This will:
- Read the font file
- Convert it to base64
- Output CSS code to copy
- Save to `opendyslexic-base64.css`

## Step 3: Update contentScript.js

Copy the generated CSS from `opendyslexic-base64.css` and add it to the `injectDyslexiaStyles()` function in `contentScript.js`.

## 🔄 Data Flow

### Curriculum Generation

```
User (Frontend)
    ↓ POST /curriculum/generate
Backend (FastAPI)
    ↓ Check cache (Supabase)
    ├─ Cache HIT → Return lesson_id
    └─ Cache MISS → LangGraph Agent
        ↓
        ├─ Node 1: Query Planner
        ├─ Node 2: Retriever
        │   ├─ YouTube Transcript API
        │   ├─ DuckDuckGo Search
        │   └─ BeautifulSoup (web scraping)
        ├─ Node 3: Synthesizer (grok-4.1-fast)
        └─ Node 4: Formatter + Citations
            ↓ Store in Supabase
            ↓ SSE Stream
User (Frontend)
    ↓ Render curriculum
    ↓ Click citation link
extensionBridge.ts
    ↓ chrome.storage.session.set()
Chrome Extension
    ↓ content_script.js
    ↓ Apply OpenDyslexic styles
External Website (styled)
```

### Socratic Sandbox

```
User (Frontend)
    ↓ Write code/text
    ↓ Click "Get a hint"
    ↓ POST /sandbox/hint
Backend (FastAPI)
    ↓ MCP Agent
    ↓ grok-3-mini (reasoning mode)
    ├─ Analyze user content
    ├─ Identify gap/error
    ├─ Generate Socratic hint
    └─ Anti-jailbreak validation
        ↓ Return hint
User (Frontend)
    ↓ Display hint
    ↓ Continue working
```

## 🗄️ Database Schema

### Supabase Tables

```
lessons
├── id (TEXT, PK)
├── goal_hash (TEXT, UNIQUE)
├── goal_raw (TEXT)
├── content (JSONB)
├── notes (TEXT)
├── sources (JSONB)
├── created_at (TIMESTAMPTZ)
└── hit_count (INTEGER)

sessions
├── id (UUID, PK)
├── created_at (TIMESTAMPTZ)
└── last_seen (TIMESTAMPTZ)

progress
├── id (UUID, PK)
├── session_id (UUID, FK → sessions)
├── lesson_id (TEXT, FK → lessons)
├── viewed_nodes (JSONB)
├── hint_counts (JSONB)
└── updated_at (TIMESTAMPTZ)
```
